# Divyansh – Personal Portfolio Website

A modern, responsive personal portfolio website built using HTML, CSS, and JavaScript.
This website showcases my skills, projects, resume, and contact information in a clean and visually engaging layout.
Designed to be simple, fast, and fully responsive across devices.

---

## Purpose of the Project

The main goals of this portfolio are:

* To present my skills, projects, and achievements in a professional way
* To make it easy for recruiters and collaborators to contact me
* To create a strong digital identity as a Computer Science Engineering student
* To practice and showcase frontend development skills
* To maintain an updated personal brand as I grow in the tech field

---

## Features

Home Section

* Dynamic role-changing text animation
* Intro banner with quick navigation buttons
* Clean gradient background and responsive layout

About Section

* Introduction about my learning journey and goals
* Details about programming languages, tools, and technologies

Projects Section

* Categorized projects: Web, Python, JavaScript, C or C Plus Plus, Java
* GitHub repository links
* Card-based responsive layout

Resume Page

* Full resume-style webpage
* Modern design using CSS Grid and Flexbox
* Includes skills, education, strengths, and project highlights

Contact Page

* Functional contact form using Web3Forms API
* Floating label style
* Clean UI with blur background

Additional Features

* Fixed navigation bar
* Smooth UI transitions
* Responsive design
* Consistent branding and footer on every page

## Deployment

This website can be deployed using GitHub Pages.

Steps for GitHub Pages

1. Push your project to a GitHub repository
2. Open Settings
3. Go to Pages
4. Select the main branch and the root folder
5. Save changes
6. The site will be live shortly
